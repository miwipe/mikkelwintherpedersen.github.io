# Mikkel Winther Pedersen – genetiker og geograf | forsker i fortidens natur, klima og mennesker

Akademisk hjemmeside af Mikkel Winther Pedersen, genetiker, geograf og forsker.
Mit arbejde fokuserer på fortidens natur, klima og menneskers aktiviteter, undersøgt gennem analyser af gammelt DNA bevaret i jord og sedimenter. Her deler jeg forskning, projekter, kode, publikationer og formidling.
